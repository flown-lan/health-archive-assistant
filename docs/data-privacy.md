# 数据与隐私政策 (Data Privacy)

PaperHealth 遵循 **隐私优先、本地优先** 的原则：

1. **本地存储**
   - 所有图片和数据存储在用户本地设备
   - SQLite 数据库 + 本地文件夹
   - 不上传任何数据到云端

2. **数据安全**
   - 数据仅在用户设备访问
   - 用户可手动备份或加密

3. **隐私承诺**
   - 项目开源，可自行验证所有数据处理逻辑
   - 不收集任何个人信息
   - 不接入第三方分析工具

4. **未来云同步（可选）**
   - 云同步功能（如果实现）必须用户主动开启
   - 云端数据将使用加密传输与存储

# PaperHealth 开发路线图 (Roadmap)

目标：将纸质病历/检验报告快速整理成个人健康档案。

## 阶段 0：基础搭建（已完成）
- 项目目录结构
- 初始代码模板
- 开源许可证 (Apache 2.0)

## 阶段 1：后端 MVP
- 文件上传接口
- OCR 模块
- 信息抽取服务
- 数据库存储
- 时间线 API
- 单条记录详情 API

## 阶段 2：前端 MVP
- 上传界面
- 时间线界面
- 详情页
- 状态提示（上传/识别状态）

## 阶段 3：本地测试与优化
- 确保本地运行
- 图片存储、OCR、数据库排序测试
- 不依赖网络

## 阶段 4：UX/UI 优化
- 界面美化
- 多文件上传
- 图片压缩/旋转
- 搜索功能
- 时间线分页/折叠

## 阶段 5：收集反馈
- 用户体验收集
- 功能优先级调整

## 阶段 6：扩展功能
- 云端同步
- 家庭档案
- AI 报告解读
- 趋势图
- PDF/JSON 导出

## 阶段 7：开源与社区
- CONTRIBUTING.md
- CODE_OF_CONDUCT.md
- 自动 CI
- 发布 v0.1 MVP
